# ComfyUI to Cinema4D Bridge

> **AI-powered creative pipeline bridging ComfyUI's generative capabilities with Cinema4D's professional 3D workflow**

## 🎯 Project Overview

Transform ideas into complete 3D scenes using natural language control. This production-ready desktop application provides:

✅ **Image Generation** - FLUX workflows with LoRA support  
✅ **3D Model Creation** - AI-powered image to 3D conversion  
✅ **Texture Generation** - PBR material creation pipeline  
✅ **Cinema4D Intelligence** - 83+ objects with natural language control  
✅ **Professional Interface** - Qt6 application with dark theme

## 🚀 Quick Start

```bash

setup_final.bat
Setup MCP Servers.bat

# 2. Configure paths in .env file
COMFYUI_PATH="D:/Comfy3D_WinPortable"
CINEMA4D_PATH="C:/Program Files/Maxon Cinema 4D 2024"

# 3. Launch application
launch.bat
```
